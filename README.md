# ⏱ Stopwatch App

A simple stopwatch built using **React**.

🔗 **Live Demo**: [Click here to view](https://stopwatch-nu-drab.vercel.app)

---

## 🚀 Features

- Start the stopwatch  
- Pause/Stop the stopwatch  
- Reset the timer to `00:00:00`  
- Clean and minimal UI  

---

## 🛠️ Tech Stack

- React  
- JavaScript  
- Vite (for fast development)  
- CSS for styling
